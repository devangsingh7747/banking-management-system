# Banking Management System (Java + MySQL)

A desktop-based Banking Management System developed using **Java Swing** and **MySQL**.  
This application simulates ATM functionality including account creation, login authentication, and banking transactions.

---

## Features

- User Signup (3-Step Account Creation)
- Secure Login using Card Number and PIN
- Automatic Card Number & PIN Generation
- Deposit Money
- Withdraw Money
- Fast Cash Withdrawal
- Balance Enquiry
- Mini Statement
- MySQL Database Integration
- Clean GUI using Java Swing

---

## Technologies Used

- Java (Core Java + Swing)
- MySQL
- JDBC (MySQL Connector)
- VS Code
- MySQL Workbench
